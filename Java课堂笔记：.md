三、数据类型于运算符

```
规则：严格区分大小写
标识符（类名）：字母、数字、下划线、美元符号。不能用数字开头，不能使用关键字
数据类型：
		基本数据类型：
			数值型（默认值为0）：
				整型：
					byte（字节型 8位 1字节 缺省值（默认值）） -128~127
					short（16位 2字节） -32768~32767
					int（32 4字节） 	-2147483648~2147483647
					long（64 8字节）
				浮点型（看到浮点类型默认为double）
					float （32 4字节）
					double（64 8字节）
			字符型（默认值为空）：
				char（2字节）：用单引号引起来
			布尔型：
				boolean（1字节）：默认值为false
		引用数据类型（复合数据类型）：
			默认值为空
			类（class）：
			接口（interface）：
			数组（array）：
局部变量：在main方法之中。必须进行初始化
全局变量：在main方法之外，需要设置为static（静态）
强制类型转换：高精度往低精度转换为强制转换（反之为自动转换）
数据类型大小：byte>short>char>int>long>float>double
double和float与计算机的硬件有关，不是一个确切值
浮点型转int：只取整数位
常量：final 数据类型 x = 值
运算符：
	赋值运算符：=
	算数运算符：
        单目运算符：++、--、
        双目运算符：+、-、
        三目运算符：？ :
	关系运算符（比较运算符）：结果为true或者false
		>、==（等于）、!=（不等于）、<=(小于等于)
		逻辑运算符：
			&(与):一假即为false
			|(或)：一真即为true
			!(非)：
			&&(短路与):只用于判断第一个表达式
			||(短路或)：上同
		复合运算符：+=、-=、*=、/=
```

六、字符串

```
字符串的比较的方法：
	s1.equals(s2):比较s1和s2的内容是否相同。
获取字符出长度的方法：
	s1.length()
	length():是方法
	length：是属性
	
数据提取方法：
	s1.substring(beginIndex,endIndex)
		beginIndex开始,endIndex结束
返回字符串指定位置上的字符：
		s1.charAt(Index)
指从字符串中，返回第一次出现的位置
	c1.indexOf():不存在就返回-1，存在就返回位置

查找方法：
	indexOf：查找第一次出现的位置
	lastindexOf：查找最后一次出现的位置
	
```
